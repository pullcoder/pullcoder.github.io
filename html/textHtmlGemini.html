텍스트 서식 예제 (Gemini 버전)

1. 텍스트 강조 및 스타일

  • 흘림체 및 굵게 (Italic Bold): 이 텍스트는 기울임과 굵게가 동시에 적용되었습니다.
  • String 강조체: "String Content"
  • Emphasized (강조/흘림): 이것은 강조된(Emphasized) 텍스트입니다.

2. 수학적 표현 (첨자)

  • 위첨자 (Superscript): 1002
  • 아래첨자 (Subscript): Log10

3. 파이썬 프로그램 코드 예제

아래는 간단한 파이썬 코드 블록입니다:

# 파이썬 예제 코드
def greet(name):
    print(f"Hello, {name}!")

# 숫자 리스트의 합계 구하기
numbers = [1, 2, 3, 4, 5]
total = sum(numbers)

print(f"합계: {total}")
greet("사용자")
