메타 태그 예제

메타 태그란?

이 문서는 HTML 문서의 구조를 나타낸 것입니다. <meta> 태그는 <head> 안에 위치하며, 브라우저와 검색 엔진에 문서 정보를 제공합니다.

메타 태그 종류

속성 (name) 값 (content) 설명
charset "UTF-8" 문서의 문자 인코딩. 한글 깨짐 방지를 위해 필수입니다.
viewport "width=device-width, initial-scale=1.0" 모바일 환경에서 화면 너비를 디바이스에 맞춥니다.
author "HTML CSS Web3D" 문서 작성자 정보입니다.
description "html5&Web3D 강좌" 검색 결과에 표시되는 페이지 설명입니다.
keyword "HTML CSS Web3D" 검색 엔진용 키워드입니다. (현재 대부분 무시됨)

코드 예제

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="author" content="HTML CSS Web3D">
    <meta name="description" content="html5&Web3D 강좌">
    <meta name="keyword" content="HTML CSS Web3D">
</head>
