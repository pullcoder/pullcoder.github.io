img 태그 주요 속성

<img src="경로" alt="설명" width="너비" height="높이" loading="eager|lazy">
  • src — 이미지 파일 경로 (필수)
  • alt — 이미지 로드 실패 시 표시할 대체 텍스트 (접근성)
  • width / height — 크기 지정 (px 단위, 생략 시 원본 크기)
  • loading="lazy" — 화면에 나타날 때 로드 (성능 최적화)
  • loading="eager" — 즉시 로드 (기본값)
