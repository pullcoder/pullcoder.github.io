HTML 문서 기본 구조

이 문서는 HTML의 기본 프레임으로만 생성된 것입니다. 모든 HTML 문서는 아래의 기본 태그들로 구성됩니다.

<!DOCTYPE html>      <!-- 문서 타입 선언 -->
<html lang="ko">    <!-- 문서 시작 -->

<head>               <!-- 문서 요약 정보 -->
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>

<body>               <!-- 화면에 표시될 내용 -->
    <p>내용</p>
</body>

</html>              <!-- 문서 끝 -->

태그별 역할

  • <!DOCTYPE html> HTML5 문서임을 브라우저에 알려주는 선언. 반드시 최상단에 위치해야 합니다.
  • <html> HTML 문서의 루트 요소. 모든 태그를 감싸며 lang 속성으로 언어를 지정합니다.
  • <head> 브라우저가 문서를 처리하는 데 필요한 메타 정보를 담습니다. 화면에 직접 표시되지 않습니다.
  • <title> 브라우저 탭과 즐겨찾기에 표시되는 문서 제목입니다.
  • <body> 사용자에게 실제로 표시되는 모든 콘텐츠(텍스트, 이미지, 링크 등)가 들어갑니다.
